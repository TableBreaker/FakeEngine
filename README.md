keys
---

###### W : move forward
###### S : move back
###### A : move left
###### D : move right
###### Q : move down
###### E : move up

###### U : rotate pitch
###### I : rotate yaw
###### O : rotate roll

###### B : both wireframe and solid
###### N : wireframe only
###### M : solid only

###### C : change object
